Gestão de Equipamentos v1 - Módulo de Equipamentos

Nesta aula, vamos desenvolver passo a passo um sistema simples de Gestão de Equipamentos.

A proposta aqui é utilizar uma classe para representar os equipamentos e, a partir disso, permitir que o usuário realize operações básicas de gerenciamento no console.

Ao longo dos requisitos, o sistema será evoluído para permitir:

  • cadastrar equipamentos;
  • editar equipamentos já registrados;
  • excluir equipamentos;
  • visualizar os equipamentos cadastrados.

Mais do que simplesmente fazer o programa funcionar, o importante nesta etapa é perceber o papel de cada trecho do código e entender como a aplicação organiza os dados em memória enquanto o programa está em execução.

Pré-Requisito: Classe Equipamento​

Esta classe servirá como a estrutura de dados que estaremos cadastrando e manipulando durante o exercício.

public class Equipamento
{
    public string id;
    public string nome;
    public string fabricante;
    public decimal precoAquisicao;
    public DateTime dataFabricacao;
}

Requisito 1: Cadastro de Equipamento​

O que o programa passa a fazer?​

Neste primeiro requisito, o sistema ganha sua funcionalidade mais básica: registrar um novo equipamento.

A ideia é permitir que o usuário informe os dados principais de um equipamento e que essas informações sejam armazenadas em uma estrutura de memória para uso posterior.

Até aqui, o foco está em aprender três coisas importantes:

  • como exibir um menu simples no console;
  • como capturar dados digitados pelo usuário;
  • como criar um objeto e armazená-lo em um array.
using System.Security.Cryptography;
using GestaoDeEquipamentos.ConsoleApp.Dominio;

Equipamento?[] equipamentos = new Equipamento[100];

while (true)
{
    Console.Clear();
    Console.WriteLine("---------------------------------");
    Console.WriteLine("Gestão de Equipamentos");
    Console.WriteLine("---------------------------------");
    Console.WriteLine("1 - Cadastrar equipamento");
    Console.WriteLine("S - Sair");
    Console.WriteLine("---------------------------------");
    Console.Write("> ");
    string? opcaoMenu = Console.ReadLine()?.ToUpper();

    if (opcaoMenu == "S")
    {
        Console.Clear();
        break;
    }

    if (opcaoMenu == "1")
    {
        Console.Clear();
        Console.WriteLine("---------------------------------");
        Console.WriteLine("Gestão de Equipamentos");
        Console.WriteLine("---------------------------------");
        Console.WriteLine("Cadastro de equipamento");
        Console.WriteLine("---------------------------------");

        string? nome;
        do
        {
            Console.Write("Digite o nome do equipamento: ");
            nome = Console.ReadLine();
        } while (string.IsNullOrWhiteSpace(nome));

        string? fabricante;
        do
        {
            Console.Write("Digite o nome do fabricante do equipamento: ");
            fabricante = Console.ReadLine();
        } while (string.IsNullOrWhiteSpace(fabricante));

        Console.Write("Digite o preço de aquisição R$ ");
        decimal precoAquisicao = Convert.ToDecimal(Console.ReadLine());

        Console.Write("Digite a data de fabricação do equipamento (dd/MM/yyyy) ");
        DateTime dataFabricacao = Convert.ToDateTime(Console.ReadLine());

        string id = Convert.ToHexString(RandomNumberGenerator.GetBytes(20))
            .ToLower()
            .Substring(0, 7);

        Equipamento equipamento = new Equipamento();
        equipamento.id = id;
        equipamento.nome = nome;
        equipamento.fabricante = fabricante;
        equipamento.precoAquisicao = precoAquisicao;
        equipamento.dataFabricacao = dataFabricacao;

        for (int i = 0; i < equipamentos.Length; i++)
        {
            if (equipamentos[i] == null)
            {
                equipamentos[i] = equipamento;
                break;
            }
        }

        Console.WriteLine("---------------------------------");
        Console.WriteLine($"O registro \"{equipamento.id}\" foi cadastrado com sucesso.");
        Console.WriteLine("---------------------------------");
        Console.Write($"Aperte ENTER para continuar...");
        Console.ReadLine();
    }
}

Como o identificador único é criado?​

Depois de capturar os dados principais, o sistema gera um identificador único:

string id = Convert.ToHexString(RandomNumberGenerator.GetBytes(20))
    .ToLower()
    .Substring(0, 7);

Aqui acontece o seguinte:

  • RandomNumberGenerator.GetBytes(20) gera 20 bytes aleatórios;
  • Convert.ToHexString(...) transforma esses bytes em texto hexadecimal;
  • ToLower() converte para letras minúsculas;
  • Substring(0, 7) mantém apenas os 7 primeiros caracteres.

Na prática, isso cria um código curto para identificar o equipamento, como se fosse um “número de registro” do item.

Requisito 2: Edição de Equipamento​

O que mudou neste requisito?​

Agora o sistema não apenas cadastra: ele também consegue alterar dados de um equipamento já existente.

Isso é importante porque, em sistemas reais, os dados nem sempre nascem corretos ou definitivos. Às vezes é preciso atualizar:

  • o nome;
  • o fabricante;
  • o preço;
  • ou a data de fabricação.
using System.Security.Cryptography;
using GestaoDeEquipamentos.ConsoleApp.Dominio;

Equipamento?[] equipamentos = new Equipamento[100];

while (true)
{
    Console.Clear();
    Console.WriteLine("---------------------------------");
    Console.WriteLine("Gestão de Equipamentos");
    Console.WriteLine("---------------------------------");
    Console.WriteLine("1 - Cadastrar equipamento");
    Console.WriteLine("2 - Editar equipamento");
    Console.WriteLine("S - Sair");
    Console.WriteLine("---------------------------------");
    Console.Write("> ");
    string? opcaoMenu = Console.ReadLine()?.ToUpper();

    if (opcaoMenu == "S")
    {
        Console.Clear();
        break;
    }

    // opções de menu anteriores omitidas
    else if (opcaoMenu == "2")
    {
        Console.Clear();
        Console.WriteLine("---------------------------------");
        Console.WriteLine("Gestão de Equipamentos");
        Console.WriteLine("---------------------------------");
        Console.WriteLine("Edição de equipamento");
        Console.WriteLine("---------------------------------");

        Console.WriteLine(
            "{0, -7} | {1, -15} | {2, -15} | {3, -22} | {4, -10}",
            "Id", "Nome", "Fabricante", "Preço de Aquisição", "Data de Fabricação"
        );

        for (int i = 0; i < equipamentos.Length; i++)
        {
            Equipamento? e = equipamentos[i];

            if (e == null)
                continue;

            Console.WriteLine(
                "{0, -7} | {1, -15} | {2, -15} | {3, -22} | {4, -10}",
                e.id, e.nome, e.fabricante, e.precoAquisicao.ToString("C2"), e.dataFabricacao.ToShortDateString()
            );
        }

        Console.WriteLine("---------------------------------");

        string? idSelecionado;
        do
        {
            Console.Write("Digite o id do equipamento que deseja editar: ");
            idSelecionado = Console.ReadLine();
        } while (string.IsNullOrWhiteSpace(idSelecionado));

        Console.WriteLine("---------------------------------");

        string? nome;
        do
        {
            Console.Write("Digite o novo nome do equipamento: ");
            nome = Console.ReadLine();
        } while (string.IsNullOrWhiteSpace(nome));

        string? fabricante;
        do
        {
            Console.Write("Digite o novo nome do fabricante do equipamento: ");
            fabricante = Console.ReadLine();
        } while (string.IsNullOrWhiteSpace(fabricante));

        Console.Write("Digite o novo preço de aquisição R$ ");
        decimal precoAquisicao = Convert.ToDecimal(Console.ReadLine());

        Console.Write("Digite a nova data de fabricação do equipamento (dd/MM/yyyy) ");
        DateTime dataFabricacao = Convert.ToDateTime(Console.ReadLine());

        Equipamento? equipamentoSelecionado = null;

        for (int i = 0; i < equipamentos.Length; i++)
        {
            Equipamento e = equipamentos[i];

            if (e == null)
                continue;

            if (e.id == idSelecionado)
            {
                equipamentoSelecionado = e;
                break;
            }
        }

        if (equipamentoSelecionado == null)
        {
            Console.WriteLine("---------------------------------");
            Console.WriteLine("Não foi possível encontrar o equipamento informado.");
            Console.WriteLine("---------------------------------");
            Console.Write("Digite ENTER para continuar...");
            continue;
        }

        equipamentoSelecionado.nome = nome;
        equipamentoSelecionado.fabricante = fabricante;
        equipamentoSelecionado.precoAquisicao = precoAquisicao;
        equipamentoSelecionado.dataFabricacao = dataFabricacao;

        Console.WriteLine("---------------------------------");
        Console.WriteLine($"O registro \"{idSelecionado}\" foi editado com sucesso.");
        Console.WriteLine("---------------------------------");
        Console.Write($"Aperte ENTER para continuar...");
        Console.ReadLine();
    }
}

O que acontece antes da edição?​

Antes de pedir os novos dados, o sistema exibe todos os equipamentos cadastrados em formato de tabela.

Isso ajuda o usuário a enxergar os registros existentes e escolher corretamente qual deseja editar.

A tabela é montada com Console.WriteLine formatado:

Console.WriteLine(
    "{0, -7} | {1, -15} | {2, -15} | {3, -22} | {4, -10}",
    "Id", "Nome", "Fabricante", "Preço de Aquisição", "Data de Fabricação"
);

Depois, o programa percorre o array para mostrar cada equipamento armazenado.

Por que isso é importante?​

Porque a edição depende de identificar corretamente qual objeto será alterado.

Em vez de editar “pela posição” do array, o programa faz isso pelo id. Isso já aproxima o sistema de uma ideia muito comum em software: cada registro possuir um identificador próprio.

Como o equipamento é localizado?​

Após mostrar a lista, o programa solicita o id do equipamento desejado.

Em seguida, percorre o array procurando um objeto cujo id seja igual ao informado:

Equipamento? equipamentoSelecionado = null;

for (int i = 0; i < equipamentos.Length; i++)
{
    Equipamento e = equipamentos[i];

    if (e == null)
        continue;

    if (e.id == idSelecionado)
    {
        equipamentoSelecionado = e;
        break;
    }
}

O que esse trecho faz?​

Ele faz uma busca simples dentro do array:

  • ignora posições vazias;
  • compara o id informado com o id de cada equipamento;
  • quando encontra o correspondente, guarda a referência em equipamentoSelecionado.

Se nenhum item for encontrado, a variável continua com valor null.

Como a edição de fato acontece?​

Depois de localizar o objeto correto, o programa substitui seus atributos pelos novos valores digitados:

equipamentoSelecionado.nome = nome;
equipamentoSelecionado.fabricante = fabricante;
equipamentoSelecionado.precoAquisicao = precoAquisicao;
equipamentoSelecionado.dataFabricacao = dataFabricacao;

Aqui não estamos criando um novo equipamento. Estamos alterando o mesmo objeto que já estava salvo no array.

Esse detalhe é importante: o sistema localiza o objeto já existente e muda seus dados diretamente.

Requisito 3: Exclusão de Equipamento​

O que o sistema aprende a fazer agora?​

Neste requisito, o sistema passa a permitir a remoção de um equipamento cadastrado.

Se no requisito anterior aprendemos a localizar um item para editar, agora usamos uma lógica parecida para remover um item da coleção.

using System.Security.Cryptography;
using GestaoDeEquipamentos.ConsoleApp.Dominio;

Equipamento?[] equipamentos = new Equipamento[100];

while (true)
{
    Console.Clear();
    Console.WriteLine("---------------------------------");
    Console.WriteLine("Gestão de Equipamentos");
    Console.WriteLine("---------------------------------");
    Console.WriteLine("1 - Cadastrar equipamento");
    Console.WriteLine("2 - Editar equipamento");
    Console.WriteLine("3 - Excluir equipamento");
    Console.WriteLine("S - Sair");
    Console.WriteLine("---------------------------------");
    Console.Write("> ");
    string? opcaoMenu = Console.ReadLine()?.ToUpper();

    if (opcaoMenu == "S")
    {
        Console.Clear();
        break;
    }

    // opções de menu anteriores omitidas
    else if (opcaoMenu == "3")
    {
        Console.Clear();
        Console.WriteLine("---------------------------------");
        Console.WriteLine("Gestão de Equipamentos");
        Console.WriteLine("---------------------------------");
        Console.WriteLine("Exclusão de equipamento");
        Console.WriteLine("---------------------------------");

        Console.WriteLine(
            "{0, -7} | {1, -15} | {2, -15} | {3, -22} | {4, -10}",
            "Id", "Nome", "Fabricante", "Preço de Aquisição", "Data de Fabricação"
        );

        for (int i = 0; i < equipamentos.Length; i++)
        {
            Equipamento? e = equipamentos[i];

            if (e == null)
                continue;

            Console.WriteLine(
                "{0, -7} | {1, -15} | {2, -15} | {3, -22} | {4, -10}",
                e.id, e.nome, e.fabricante, e.precoAquisicao.ToString("C2"), e.dataFabricacao.ToShortDateString()
            );
        }

        Console.WriteLine("---------------------------------");

        string? idSelecionado;
        do
        {
            Console.Write("Digite o id do equipamento que deseja excluir: ");
            idSelecionado = Console.ReadLine();
        } while (string.IsNullOrWhiteSpace(idSelecionado));

        bool equipamentoExcluido = false;

        for (int i = 0; i < equipamentos.Length; i++)
        {
            Equipamento? e = equipamentos[i];

            if (e == null)
                continue;

            if (e.id == idSelecionado)
            {
                equipamentos[i] = null;
                equipamentoExcluido = true;
                break;
            }
        }

        if (equipamentoExcluido)
        {
            Console.WriteLine("---------------------------------");
            Console.WriteLine($"O registro \"{idSelecionado}\" foi excluído com sucesso.");
            Console.WriteLine("---------------------------------");
        }
        else
        {
            Console.WriteLine("---------------------------------");
            Console.WriteLine($"Não foi possível encontrar o registro \"{idSelecionado}\".");
            Console.WriteLine("---------------------------------");
        }

        Console.Write($"Aperte ENTER para continuar...");
        Console.ReadLine();
    }
}

Como a exclusão começa?​

Assim como na edição, o sistema primeiro exibe a listagem dos equipamentos cadastrados.

Isso é útil porque o usuário precisa enxergar os registros disponíveis antes de escolher qual deseja excluir.

Depois, o programa pede o id do equipamento a ser removido.

Como a remoção é feita?​

O código percorre o array em busca do equipamento cujo id seja igual ao digitado:

for (int i = 0; i < equipamentos.Length; i++)
{
    Equipamento? e = equipamentos[i];

    if (e == null)
        continue;

    if (e.id == idSelecionado)
    {
        equipamentos[i] = null;
        break;
    }
}

O que significa colocar null naquela posição?​

Significa que o espaço antes ocupado por aquele equipamento volta a ficar vazio.

Na prática, o objeto deixa de estar acessível naquela coleção, e a posição poderá ser reutilizada depois por um novo cadastro.

Ou seja, a exclusão aqui não “encolhe” o array. O array continua com 100 posições, mas uma delas volta a ficar desocupada.

Requisito 4: Visualização de Equipamentos​

O que é acrescentado aqui?​

Neste último requisito, o sistema passa a ter uma opção específica para consultar os equipamentos cadastrados sem editar e sem excluir.

Essa funcionalidade é importante porque, em muitos sistemas, o usuário quer apenas visualizar os dados disponíveis.

using System.Security.Cryptography;
using GestaoDeEquipamentos.ConsoleApp.Dominio;

Equipamento?[] equipamentos = new Equipamento[100];

while (true)
{
    Console.Clear();
    Console.WriteLine("---------------------------------");
    Console.WriteLine("Gestão de Equipamentos");
    Console.WriteLine("---------------------------------");
    Console.WriteLine("1 - Cadastrar equipamento");
    Console.WriteLine("2 - Editar equipamento");
    Console.WriteLine("3 - Excluir equipamento");
    Console.WriteLine("4 - Visualizar equipamentos");
    Console.WriteLine("S - Sair");
    Console.WriteLine("---------------------------------");
    Console.Write("> ");
    string? opcaoMenu = Console.ReadLine()?.ToUpper();

    if (opcaoMenu == "S")
    {
        Console.Clear();
        break;
    }

    // opções de menu anteriores omitidas
    else if (opcaoMenu == "4")
    {
        Console.Clear();
        Console.WriteLine("---------------------------------");
        Console.WriteLine("Gestão de Equipamentos");
        Console.WriteLine("---------------------------------");
        Console.WriteLine("Visualização de equipamentos");
        Console.WriteLine("---------------------------------");

        Console.WriteLine(
            "{0, -7} | {1, -15} | {2, -15} | {3, -22} | {4, -10}",
            "Id", "Nome", "Fabricante", "Preço de Aquisição", "Data de Fabricação"
        );

        for (int i = 0; i < equipamentos.Length; i++)
        {
            Equipamento? e = equipamentos[i];

            if (e == null)
                continue;

            Console.WriteLine(
                "{0, -7} | {1, -15} | {2, -15} | {3, -22} | {4, -10}",
                e.id, e.nome, e.fabricante, e.precoAquisicao.ToString("C2"), e.dataFabricacao.ToShortDateString()
            );
        }

        Console.WriteLine("---------------------------------");
        Console.Write($"Aperte ENTER para continuar...");
        Console.ReadLine();
    }
}

O que acontece ao visualizar?​

Quando o usuário escolhe a opção "4", o sistema:

  1. limpa a tela;
  2. mostra o cabeçalho da funcionalidade;
  3. imprime a tabela com os dados dos equipamentos;
  4. aguarda o usuário pressionar ENTER para voltar ao menu.

O processo de exibição é praticamente o mesmo usado nos requisitos anteriores para apoiar a edição e a exclusão.

Por que isso é importante?​

Porque aqui a listagem deixa de ser apenas um apoio para outra operação e passa a ser uma funcionalidade própria do sistema.

Ou seja, o usuário pode simplesmente consultar o que já foi cadastrado.

O que esse requisito consolida?​

Este requisito fecha a primeira versão do sistema mostrando que já existe um pequeno CRUD em console:

  • Create → cadastrar;
  • Read → visualizar;
  • Update → editar;
  • Delete → excluir.

Mesmo que ainda seja uma implementação simples, ela já permite trabalhar com o ciclo completo de manipulação de dados em memória.
