Separação de Responsabilidades

Nesta aula, vamos evoluir um programa que possui responsabilidades misturadas para uma versão mais organizada, utilizando o conceito de separação de responsabilidades.

Nosso objetivo é:

  • Organizar melhor o código
  • Facilitar manutenção
  • Tornar o sistema mais claro de entender
  • Preparar o terreno para aplicações maiores

Problema Inicial: Responsabilidades Misturadas​

O que está acontecendo aqui?

No código original, a classe ContaCorrente está fazendo muitas coisas ao mesmo tempo:

  • Lógica de negócio (saldo, saque, depósito)
  • Entrada de dados (Console.ReadLine)
  • Exibição de mensagens (Console.WriteLine)

Ou seja, ela está acumulando mais de uma responsabilidade.

Isso dificulta:

  • Reutilização do código
  • Manutenção futura
  • Testes

Exemplo do problema

public void Sacar()
{
    Console.Write("Digite o valor que deseja sacar: ");
    decimal valor = Convert.ToDecimal(Console.ReadLine());

    if (valor > saldo + limiteDebito)
        Console.WriteLine("Limite excedido!");
    else
        saldo -= valor;
}

Por que isso é um problema?​

Porque a classe ContaCorrente deveria apenas cuidar da lógica da conta, e não da interface com o usuário.

Requisito 1: Separar responsabilidades​

O que vamos fazer?

Vamos dividir o sistema em duas partes principais:

  • ContaCorrente → regras de negócio
  • TelaPrincipal → interação com o usuário

Cada classe passa a ter uma responsabilidade clara.

Requisito 2: Refatorar a classe ContaCorrente​

Antes​

A classe fazia tudo (lógica + interface)

Depois​

Ela passa a trabalhar apenas com dados e regras

class ContaCorrente
{
    public int numeroIdentificacao;
    public string titular;
    public decimal saldo;
    public decimal limiteDebito;

    public bool Sacar(decimal valorSaque)
    {
        if (valorSaque > saldo + limiteDebito)
            return false;
        else
            saldo -= valorSaque;

        return true;
    }

    public void Depositar(decimal valorDeposito)
    {
        saldo += valorDeposito;
    }

    public bool TransferirPara(ContaCorrente contaDestino, decimal valorTransferencia)
    {
        bool saqueConcluido = Sacar(valorTransferencia);

        if (!saqueConcluido)
            return false;

        contaDestino.saldo += valorTransferencia;

        return true;
    }

    public decimal ObterSaldo()
    {
        return saldo;
    }
}

O que mudou?​

  • Não usa mais Console
  • Recebe valores como parâmetro
  • Retorna resultado (true ou false)

Por que isso é importante?​

Agora a classe:

  • Pode ser reutilizada em qualquer contexto (console, web, API…)
  • Fica mais fácil de testar
  • Tem responsabilidade única

Requisito 3: Criar a classe TelaPrincipal​

O que estamos fazendo?​

Toda a interação com o usuário agora fica em uma nova classe:

class TelaPrincipal
{
    public string? ApresentarMenuOpcoes(ContaCorrente contaAcessada)
    {
        Console.Clear();
        Console.WriteLine("-------------------------------------");
        Console.WriteLine($"Conta Corrente #{contaAcessada.numeroIdentificacao} de {contaAcessada.titular}");
        Console.WriteLine("-------------------------------------");
        Console.WriteLine("1 - Saque");
        Console.WriteLine("2 - Depósito");
        Console.WriteLine("3 - Transferência");
        Console.WriteLine("4 - Consulta de Saldo");
        Console.WriteLine("S - Sair");
        Console.WriteLine("-------------------------------------");
        Console.Write("Digite uma opção válida: ");
        string? opcaoMenu = Console.ReadLine()?.ToUpper();

        return opcaoMenu;
    }

    public void ApresentarOperacaoSaque(ContaCorrente contaAcessada)
    {
        Console.WriteLine("-------------------------------------");
        Console.Write("Digite o valor que deseja sacar (R$): ");
        decimal valorSaque = Convert.ToDecimal(Console.ReadLine());

        bool saqueConcluido = contaAcessada.Sacar(valorSaque);

        if (saqueConcluido)
        {
            Console.WriteLine("-------------------------------------");
            Console.WriteLine("O valor foi sacado com sucesso!");
        }
        else
        {
            Console.WriteLine("-------------------------------------");
            Console.WriteLine("O valor do limite de débito foi ultrapassado!");
        }

        Console.WriteLine("-------------------------------------");
        Console.Write("Digite ENTER para continuar...");
        Console.ReadLine();
    }

    public void ApresentarOperacaoDeposito(ContaCorrente contaAcessada)
    {
        Console.WriteLine("-------------------------------------");
        Console.Write("Digite o valor que deseja depositar (R$): ");
        decimal valorDeposito = Convert.ToDecimal(Console.ReadLine());

        contaAcessada.Depositar(valorDeposito);

        Console.WriteLine("-------------------------------------");
        Console.WriteLine("O valor foi depositado com sucesso!");
        Console.WriteLine("-------------------------------------");
        Console.Write("Digite ENTER para continuar...");
        Console.ReadLine();
    }

    public void ApresentarOperacaoTransferencia(
        ContaCorrente contaAcessada, ContaCorrente contaDestino)
    {
        Console.WriteLine("-------------------------------------");
        Console.Write("Digite o valor que deseja transferir (R$): ");
        decimal valorTransferencia = Convert.ToDecimal(Console.ReadLine());

        contaAcessada.TransferirPara(contaDestino, valorTransferencia);

        Console.WriteLine("-------------------------------------");
        Console.WriteLine($"O valor de R${valorTransferencia} foi tranferido com sucesso!");
        Console.WriteLine("-------------------------------------");
        Console.Write("Digite ENTER para continuar...");
        Console.ReadLine();
    }

    public void ApresentarOperacaoObterSaldo(ContaCorrente contaAcessada)
    {
        decimal saldo = contaAcessada.ObterSaldo();

        Console.WriteLine("-------------------------------------");
        Console.WriteLine("O valor do saldo da conta é de (R$): " + saldo);
        Console.WriteLine("-------------------------------------");
        Console.Write("Digite ENTER para continuar...");
        Console.ReadLine();
    }
}

Responsabilidade da TelaPrincipal​

  • Mostrar menus
  • Ler entradas do usuário
  • Exibir mensagens

Ou seja: interface do sistema

Requisito 4: Conectar as responsabilidades​

Agora a TelaPrincipal chama a ContaCorrente:

class Program
{
    static void Main(string[] args)
    {
        // Conta Corrente 1
        ContaCorrente contaUm = new ContaCorrente();
        contaUm.numeroIdentificacao = 1;
        contaUm.titular = "Tiago";
        contaUm.saldo = 400;
        contaUm.limiteDebito = 1200;

        // Conta Corrente 2
        ContaCorrente contaDois = new ContaCorrente();
        contaDois.numeroIdentificacao = 2;
        contaDois.titular = "Rech";
        contaDois.saldo = 12000;
        contaDois.limiteDebito = 1200;

        TelaPrincipal tela = new TelaPrincipal();

        ContaCorrente contaAcessada = contaUm;

        while (true)
        {
            string? opcaoMenu = tela.ApresentarMenuOpcoes(contaAcessada);

            if (opcaoMenu == "S")
                break;

            if (opcaoMenu == "1")
                tela.ApresentarOperacaoSaque(contaAcessada);

            else if (opcaoMenu == "2")
                tela.ApresentarOperacaoDeposito(contaAcessada);

            else if (opcaoMenu == "3")
                tela.ApresentarOperacaoTransferencia(contaAcessada, contaDois);

            else if (opcaoMenu == "4")
                tela.ApresentarOperacaoObterSaldo(contaAcessada);
        }
    }
}
