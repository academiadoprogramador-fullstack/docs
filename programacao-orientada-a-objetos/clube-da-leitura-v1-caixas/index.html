Clube da Leitura v1 - Caixas de Revistas

Nesta etapa, vamos iniciar o desenvolvimento do sistema de Clube da Leitura com foco em dois conceitos fundamentais:

  • organização em camadas
  • encapsulamento no domínio

Agora, vamos entender melhor como o sistema funcionará.

Como o sistema está organizado​

O sistema segue o mesmo modelo já utilizado anteriormente:

Domínio​

Representa os dados e regras do sistema.

Infraestrutura​

Responsável por armazenar e recuperar dados.

Apresentação​

Responsável por interagir com o usuário.

O que acontece entre as camadas?​

Antes de entrar no código, vamos entender o fluxo completo:

Exemplo: Cadastro de Caixa​

  1. O usuário escolhe cadastrar na tela
  2. A tela coleta os dados
  3. A tela cria um objeto do domínio
  4. O domínio valida seus próprios dados
  5. A tela envia o objeto para o repositório
  6. O repositório armazena o objeto

👉 Fluxo:

Usuário → Tela → Domínio → Repositório → Memória

Por que isso é importante?​

Cada camada tem uma responsabilidade clara:

  • a tela não valida regras complexas
  • o domínio não sabe nada sobre console
  • o repositório não interage com o usuário

👉 Isso evita código confuso e acoplado.

Requisito 1: Classse do Domínio Caixa​

public class Caixa
{
    public string Id { get; set; } = string.Empty;
    public string Etiqueta { get; set; } = string.Empty;
    public string Cor { get; set; } = string.Empty;
    public int TempoEmprestimo { get; set; }

    public Caixa(string etiqueta, string cor, int tempoEmprestimo)
    {
        Id = Convert
                .ToHexString(RandomNumberGenerator.GetBytes(20))
                .ToLower()
                .Substring(0, 7);

        Etiqueta = etiqueta;
        Cor = cor;
        TempoEmprestimo = tempoEmprestimo;
    }
}

Encapsulamento na prática​

Encapsulamento significa:

👉 a classe controla seus próprios dados

Métodos adicionais​

public class Caixa
{
    // ...propriedades e construtor omitidos

    public string[] Validar()
    {
        string erros = string.Empty;

        if (string.IsNullOrEmpty(Cor.Trim()))
            erros += "O campo \"Cor\" é obrigatório;";

        if (string.IsNullOrEmpty(Etiqueta.Trim()))
            erros += "O campo \"Etiqueta\" é obrigatório;";

        if (TempoEmprestimo < 1)
            erros += "O campo \"Tempo de Empréstimo\" é obrigatório;";


        return erros.Split(';', StringSplitOptions.RemoveEmptyEntries);
    }

    public void AtualizarRegistro(Caixa registroAtualizado)
    {
        Etiqueta = registroAtualizado.Etiqueta;
        Cor = registroAtualizado.Cor;
        TempoEmprestimo = registroAtualizado.TempoEmprestimo;
    }
}

Validação dentro da classe​

public string[] Validar()
{
    string erros = string.Empty;

    if (string.IsNullOrEmpty(Cor.Trim()))
        erros += "O campo \"Cor\" é obrigatório;";

    if (string.IsNullOrEmpty(Etiqueta.Trim()))
        erros += "O campo \"Etiqueta\" é obrigatório;";

    if (TempoEmprestimo < 1)
        erros += "O campo \"Tempo de Empréstimo\" é obrigatório;";


    return erros.Split(';', StringSplitOptions.RemoveEmptyEntries);
}

O que acontece aqui?​

A própria classe verifica:

  • se a etiqueta foi informada
  • se a cor é válida
  • se o tempo é maior que zero

👉 Isso evita espalhar validações pelo sistema.

Atualização controlada​

public void AtualizarRegistro(Caixa registroAtualizado)
{
    Etiqueta = registroAtualizado.Etiqueta;
    Cor = registroAtualizado.Cor;
    TempoEmprestimo = registroAtualizado.TempoEmprestimo;
}

O que isso resolve?​

Em vez de fazer isso na tela ou repositório:

caixa.Etiqueta = nova.Etiqueta;
caixa.Cor = nova.Cor;

Agora fazemos:

caixa.AtualizarRegistro(nova);

👉 A própria classe decide como atualizar seus dados.

Por que isso é importante?​

Isso garante que:

  • alterações sejam controladas
  • regras sejam centralizadas
  • o objeto nunca fique inconsistente

👉 Isso é encapsulamento.

Requisito 2: Acesso à dados com RepositorioCaixa​

O que essa camada faz?​

Ela cuida apenas de:

  • armazenar caixas
  • buscar caixas
  • editar
  • excluir
public class RepositorioCaixa
{
    private Caixa?[] caixas = new Caixa[100];

    public void Cadastrar(Caixa novaCaixa)
    {
        for (int i = 0; i < caixas.Length; i++)
        {
            Caixa? c = caixas[i];

            if (c == null)
            {
                caixas[i] = novaCaixa;
                break;
            }
        }
    }

    public bool Editar(string idSelecionado, Caixa novaCaixa)
    {
        Caixa? caixaSelecionada = SelecionarPorId(idSelecionado);

        if (caixaSelecionada == null)
            return false;

        caixaSelecionada.AtualizarRegistro(novaCaixa);

        return true;
    }

    public bool Excluir(Caixa caixaSelecionada)
    {
        for (int i = 0; i < caixas.Length; i++)
        {
            Caixa? c = caixas[i];

            if (c == null)
                continue;

            if (c == caixaSelecionada)
            {
                caixas[i] = null;
                return true;
            }
        }

        return false;
    }

    public bool Excluir(string idSelecionado)
    {
        for (int i = 0; i < caixas.Length; i++)
        {
            Caixa? c = caixas[i];

            if (c == null)
                continue;

            if (c.Id == idSelecionado)
            {
                caixas[i] = null;
                return true;
            }
        }

        return false;
    }

    public Caixa? SelecionarPorId(string idSelecionado)
    {
        Caixa? caixaSelecionado = null;

        for (int i = 0; i < caixas.Length; i++)
        {
            Caixa? c = caixas[i];

            if (c == null)
                continue;

            if (c.Id == idSelecionado)
            {
                caixaSelecionado = c;
                break;
            }
        }

        return caixaSelecionado;
    }

    public Caixa?[] SelecionarTodos()
    {
        return caixas;
    }
}

Comunicação com o domínio​

Veja o método de edição:

caixaSelecionada.AtualizarRegistro(novaCaixa);

👉 O repositório não altera propriedades diretamente 👉 Ele delega ao domínio

O que isso significa?​

  • o repositório não conhece regras de negócio
  • ele apenas executa operações

👉 Isso mantém o sistema organizado.

Requisito 3: Apresentação com TelaCaixa​

O que essa camada faz?​

  • exibe menus
  • coleta dados
  • mostra mensagens
  • chama o repositório
public class TelaCaixa
{
    private RepositorioCaixa repositorioCaixa;

    public TelaCaixa(RepositorioCaixa rC)
    {
        repositorioCaixa = rC;
    }

    public string? ObterEscolhaMenuPrincipal()
    {
        Console.Clear();
        Console.WriteLine("---------------------------------");
        Console.WriteLine("Gestão de Caixas");
        Console.WriteLine("---------------------------------");
        Console.WriteLine("1 - Cadastrar caixa");
        Console.WriteLine("2 - Editar caixa");
        Console.WriteLine("3 - Excluir caixa");
        Console.WriteLine("4 - Visualizar caixas");
        Console.WriteLine("S - Voltar para o início");
        Console.WriteLine("---------------------------------");
        Console.Write("> ");
        string? opcaoMenu = Console.ReadLine()?.ToUpper();

        return opcaoMenu;
    }

    public void Cadastrar()
    {
        Notificador.ExibirCabecalho("Cadastro de Caixa");

        Caixa novaCaixa = ObterDadosCadastrais();

        string[] errosValidacao = novaCaixa.Validar();

        if (errosValidacao.Length > 0)
        {
            Console.ForegroundColor = ConsoleColor.Red;

            for (int i = 0; i < errosValidacao.Length; i++)
                Console.WriteLine(errosValidacao[i]);

            Console.ResetColor();

            Console.WriteLine("---------------------------------");
            Console.WriteLine("Digite ENTER para continuar...");
            Console.ReadLine();

            Cadastrar();
            return;
        }

        repositorioCaixa.Cadastrar(novaCaixa);

        Notificador.ExibirMensagem($"O registro \"{novaCaixa.Id}\" foi cadastrado com sucesso.");
    }

    public void Editar()
    {
        Notificador.ExibirCabecalho("Edição de Caixa");

        VisualizarTodos(deveExibirCabecalho: false);

        Console.WriteLine("---------------------------------");

        string? idSelecionado;

        do
        {
            Console.Write("Digite o id do registro que deseja editar: ");
            idSelecionado = Console.ReadLine();

            if (!string.IsNullOrWhiteSpace(idSelecionado) && idSelecionado.Length == 7)
                break;
        } while (true);

        Caixa novaCaixa = ObterDadosCadastrais();

        bool conseguiuEditar = repositorioCaixa.Editar(idSelecionado, novaCaixa);

        if (!conseguiuEditar)
        {
            Notificador.ExibirMensagem($"Não foi possível encontrar o registro informado.");
            return;
        }

        Notificador.ExibirMensagem($"O registro \"{idSelecionado}\" foi editado com sucesso.");
    }

    public void Excluir()
    {
        Notificador.ExibirCabecalho("Exclusão de Caixa");

        VisualizarTodos(deveExibirCabecalho: false);

        Console.WriteLine("---------------------------------");

        string? idSelecionado;

        do
        {
            Console.Write("Digite o id do registro que deseja excluir: ");
            idSelecionado = Console.ReadLine();

            if (!string.IsNullOrWhiteSpace(idSelecionado) && idSelecionado.Length == 7)
                break;
        } while (true);

        bool conseguiuExcluir = repositorioCaixa.Excluir(idSelecionado);

        if (!conseguiuExcluir)
        {
            Notificador.ExibirMensagem($"Não foi possível encontrar o registro informado.");
            return;
        }

        Notificador.ExibirMensagem($"O registro \"{idSelecionado}\" foi excluído com sucesso.");
    }

    public void VisualizarTodos(bool deveExibirCabecalho)
    {
        if (deveExibirCabecalho)
            Notificador.ExibirCabecalho("Visualização de Caixas");

        Console.WriteLine(
            "{0, -7} | {1, -20} | {2, -10} | {3, -20}",
            "Id", "Etiqueta", "Cor", "Tempo de Empréstimo"
        );

        Caixa?[] caixas = repositorioCaixa.SelecionarTodos();

        for (int i = 0; i < caixas.Length; i++)
        {
            Caixa? c = caixas[i];

            if (c == null)
                continue;

            string corSelecionada = c.Cor;

            if (corSelecionada == "Vermelha")
                Console.ForegroundColor = ConsoleColor.Red;

            else if (corSelecionada == "Verde")
                Console.ForegroundColor = ConsoleColor.Green;

            else if (corSelecionada == "Azul")
                Console.ForegroundColor = ConsoleColor.Blue;

            Console.WriteLine(
                "{0, -7} | {1, -20} | {2, -10} | {3, -20}",
                c.Id, c.Etiqueta, c.Cor, c.TempoEmprestimo
            );

            Console.ResetColor();
        }

        if (deveExibirCabecalho)
        {
            Console.WriteLine("---------------------------------");
            Console.Write("Digite ENTER para continuar...");
            Console.ReadLine();
        }
    }

    public Caixa ObterDadosCadastrais()
    {
        Console.Write("Informe a etiqueta da caixa: ");
        string? etiqueta = Console.ReadLine();

        Console.WriteLine("---------------------------------");
        Console.WriteLine("Selecione uma cor válida para a caixa");
        Console.ForegroundColor = ConsoleColor.Red;
        Console.WriteLine("1 - Vermelha");
        Console.ForegroundColor = ConsoleColor.Green;
        Console.WriteLine("2 - Verde");
        Console.ForegroundColor = ConsoleColor.Blue;
        Console.WriteLine("3 - Azul");
        Console.ResetColor();
        Console.WriteLine("4 - Branca (Padrão)");
        Console.WriteLine("---------------------------------");

        Console.Write("Informe a cor da caixa: ");
        string? codigoCorSelecionada = Console.ReadLine();

        string corSelecionada;

        if (codigoCorSelecionada == "1")
            corSelecionada = "Vermelha";

        else if (codigoCorSelecionada == "2")
            corSelecionada = "Verde";

        else if (codigoCorSelecionada == "3")
            corSelecionada = "Azul";

        else
            corSelecionada = "Branca";

        Console.WriteLine("---------------------------------");

        Console.Write("Informe o tempo de empréstimo de revistas da caixa: ");
        int tempoEmprestimo = Convert.ToInt32(Console.ReadLine());

        return new Caixa(etiqueta, corSelecionada, tempoEmprestimo);
    }
}

Validação​

Caixa novaCaixa = ObterDadosCadastrais();

string[] erros = novaCaixa.Validar();

👉 Aqui acontece algo importante:

A tela chama o domínio para validar

Tratamento de erro​

if (errosValidacao.Length > 0)

👉 A tela decide o que fazer com os erros

  • mostrar mensagem
  • pedir novamente

Separação de responsabilidades​

  • domínio → valida
  • tela → decide como reagir

👉 Isso evita mistura de responsabilidades.

Como tudo se conecta?​

Na Program:

RepositorioCaixa repositorioCaixa = new RepositorioCaixa();
TelaCaixa telaCaixa = new TelaCaixa(repositorioCaixa);

O que isso significa?​

  • a tela depende do repositório
  • o repositório trabalha com o domínio

👉 Temos um fluxo encadeado:

Tela → Repositório → Domínio

O papel do encapsulamento nesse fluxo​

Vamos observar o caminho dos dados:

  1. A tela cria o objeto
  2. O domínio valida
  3. O repositório armazena
  4. O domínio controla alterações

👉 O domínio está no centro de tudo
