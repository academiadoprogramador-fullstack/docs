Nesta etapa, vamos evoluir o sistema de Clube da Leitura adicionando um novo módulo: **Revistas**.

Na versão anterior, o sistema já era capaz de gerenciar caixas, utilizando:

- separação em camadas;
- encapsulamento no domínio;
- operações básicas de CRUD.

Agora, vamos dar um passo além: trabalhar com **mais de uma entidade** e introduzir relacionamentos entre objetos.

## O que muda nesta versão?

Até agora, o sistema lidava com apenas um tipo de informação:

- caixas

Agora, passamos a lidar também com:

- revistas

## O que é uma revista no sistema?

Uma revista representa um item que será armazenado dentro de uma caixa.

Ou seja:

👉 toda revista precisa estar associada a uma caixa

## Relação entre as classes

Isso significa que agora temos um relacionamento:

- uma `Caixa` pode ter várias revistas
- uma `Revista` pertence a uma única caixa

👉 Esse é o primeiro momento em que o sistema começa a trabalhar com **objetos conectados entre si**

---

## Requisito 1: Classe de Domínio `Revista`

### O que o sistema passa a representar?

Agora criamos uma nova classe:

```cs
public class Revista
{
    public string Id { get; set; } = string.Empty;
    public string Titulo { get; set; } = string.Empty;
    public int NumeroEdicao { get; set; }
    public int Ano { get; set; }
    public Caixa Caixa { get; set; }

    public Revista(string titulo, int numeroEdicao, int ano, Caixa caixa)
    {
        Id = Convert
                .ToHexString(RandomNumberGenerator.GetBytes(20))
                .ToLower()
                .Substring(0, 7);

        Titulo = titulo;
        NumeroEdicao = numeroEdicao;
        Ano = ano;
        Caixa = caixa;
    }
}
```

### O que essa classe representa?

Uma revista possui:

- título
- número da edição
- ano
- a caixa onde está armazenada

👉 Diferente da `Caixa`, agora temos uma classe que **depende de outra classe**

### Encapsulamento no domínio

Assim como na `Caixa`, a classe `Revista` também controla seus próprios dados.

#### Validação

```cs

public string[] Validar()
{
    string erros = string.Empty;

    if (string.IsNullOrEmpty(Titulo.Trim()))
        erros += "O campo \"Titulo\" é obrigatório;";

    if (NumeroEdicao < 0)
        erros += "O campo \"Número da Edição\" precisa ser maior ou igual a zero;";

    if (Ano < 1)
        erros += "O campo \"Ano\" precisa ser válido;";

    if (Caixa == null)
        erros += "O campo \"Caixa\" é obrigatório;";

    return erros.Split(';', StringSplitOptions.RemoveEmptyEntries);
}
```

#### O que acontece aqui?

A própria classe verifica:

- se o título foi informado
- se o número da edição é válido
- se o ano é válido
- se a caixa foi selecionada

👉 Isso evita validações espalhadas pela aplicação

#### Atualização controlada

```cs
public void AtualizarRegistro(Revista registroAtualizado)
{
    Titulo = registroAtualizado.Titulo;
    NumeroEdicao = registroAtualizado.NumeroEdicao;
    Ano = registroAtualizado.Ano;
    Caixa = registroAtualizado.Caixa;
}
```

👉 Em vez de alterar atributos diretamente, a própria classe decide como atualizar seus dados.

#### Por que isso é importante?

Isso garante:

- consistência dos dados
- centralização de regras
- maior segurança na manipulação do objeto

## Requisito 02: Acesso à dados com `RepositorioRevista`

### O que o sistema aprende a fazer?

Criamos:

```cs
public class RepositorioRevista
{
    private Revista?[] revistas = new Revista[100];

    public void Cadastrar(Revista novaRevista)
    {
        for (int i = 0; i < revistas.Length; i++)
        {
            Revista? r = revistas[i];

            if (r == null)
            {
                revistas[i] = novaRevista;
                break;
            }
        }
    }

    public bool Editar(string idSelecionado, Revista novaRevista)
    {
        Revista? revistaSelecionada = SelecionarPorId(idSelecionado);

        if (revistaSelecionada == null)
            return false;

        revistaSelecionada.AtualizarRegistro(novaRevista);

        return true;
    }

    public bool Excluir(Revista revistaSelecionada)
    {
        for (int i = 0; i < revistas.Length; i++)
        {
            Revista? r = revistas[i];

            if (r == null)
                continue;

            if (r == revistaSelecionada)
            {
                revistas[i] = null;
                return true;
            }
        }

        return false;
    }

    public bool Excluir(string idSelecionado)
    {
        for (int i = 0; i < revistas.Length; i++)
        {
            Revista? r = revistas[i];

            if (r == null)
                continue;

            if (r.Id == idSelecionado)
            {
                revistas[i] = null;
                return true;
            }
        }

        return false;
    }

    public Revista? SelecionarPorId(string idSelecionado)
    {
        Revista? revistaSelecionada = null;

        for (int i = 0; i < revistas.Length; i++)
        {
            Revista? r = revistas[i];

            if (r == null)
                continue;

            if (r.Id == idSelecionado)
            {
                revistaSelecionada = r;
                break;
            }
        }

        return revistaSelecionada;
    }

    public Revista?[] SelecionarTodos()
    {
        return revistas;
    }
}
```

### Responsabilidade

Esse repositório é responsável por:

- armazenar revistas
- cadastrar
- editar
- excluir
- buscar por id
- listar todas

### Integração com o domínio

Veja o método de edição:

```cs
revistaSelecionada.AtualizarRegistro(novaRevista);
```

- 👉 O repositório não altera diretamente os atributos
- 👉 Ele delega ao domínio

### O que isso significa?

- o repositório não contém regras de negócio
- ele apenas executa operações

👉 Isso mantém o código organizado e previsível

## Requisito 03: Apresentação com `TelaRevista`

### O que essa camada faz?

- exibe menus
- coleta dados do usuário
- chama o repositório
- interage com caixas

```cs
public class TelaRevista
{
    private RepositorioRevista repositorioRevista;
    private RepositorioCaixa repositorioCaixa;

    public TelaRevista(RepositorioRevista rR, RepositorioCaixa rC)
    {
        repositorioRevista = rR;
        repositorioCaixa = rC;
    }

    public string? ObterEscolhaMenuPrincipal()
    {
        Console.Clear();
        Console.WriteLine("---------------------------------");
        Console.WriteLine("Gestão de Revistas");
        Console.WriteLine("---------------------------------");
        Console.WriteLine("1 - Cadastrar revista");
        Console.WriteLine("2 - Editar revista");
        Console.WriteLine("3 - Excluir revista");
        Console.WriteLine("4 - Visualizar revistas");
        Console.WriteLine("S - Voltar para o início");
        Console.WriteLine("---------------------------------");
        Console.Write("> ");
        string? opcaoMenu = Console.ReadLine()?.ToUpper();

        return opcaoMenu;
    }

    public void Cadastrar()
    {
        Notificador.ExibirCabecalho("Cadastro de Revista");

        Revista novaRevista = ObterDadosCadastrais();

        string[] errosValidacao = novaRevista.Validar();

        if (errosValidacao.Length > 0)
        {
            Console.ForegroundColor = ConsoleColor.Red;

            for (int i = 0; i < errosValidacao.Length; i++)
                Console.WriteLine(errosValidacao[i]);

            Console.ResetColor();

            Console.WriteLine("---------------------------------");
            Console.WriteLine("Digite ENTER para continuar...");
            Console.ReadLine();

            Cadastrar();
            return;
        }

        repositorioRevista.Cadastrar(novaRevista);

        Notificador.ExibirMensagem($"O registro \"{novaRevista.Id}\" foi cadastrado com sucesso.");
    }

    public void Editar()
    {
        Notificador.ExibirCabecalho("Edição de Revista");

        VisualizarTodos(deveExibirCabecalho: false);

        Console.WriteLine("---------------------------------");

        string? idSelecionado;

        do
        {
            Console.Write("Digite o id do registro que deseja editar: ");
            idSelecionado = Console.ReadLine();

            if (!string.IsNullOrWhiteSpace(idSelecionado) && idSelecionado.Length == 7)
                break;
        } while (true);

        Revista novaRevista = ObterDadosCadastrais();

        bool conseguiuEditar = repositorioRevista.Editar(idSelecionado, novaRevista);

        if (!conseguiuEditar)
        {
            Notificador.ExibirMensagem($"Não foi possível encontrar o registro informado.");
            return;
        }

        Notificador.ExibirMensagem($"O registro \"{idSelecionado}\" foi editado com sucesso.");
    }

    public void Excluir()
    {
        Notificador.ExibirCabecalho("Exclusão de Revista");

        VisualizarTodos(deveExibirCabecalho: false);

        Console.WriteLine("---------------------------------");

        string? idSelecionado;

        do
        {
            Console.Write("Digite o id do registro que deseja excluir: ");
            idSelecionado = Console.ReadLine();

            if (!string.IsNullOrWhiteSpace(idSelecionado) && idSelecionado.Length == 7)
                break;
        } while (true);

        bool conseguiuExcluir = repositorioRevista.Excluir(idSelecionado);

        if (!conseguiuExcluir)
        {
            Notificador.ExibirMensagem($"Não foi possível encontrar o registro informado.");
            return;
        }

        Notificador.ExibirMensagem($"O registro \"{idSelecionado}\" foi excluído com sucesso.");
    }

    public void VisualizarTodos(bool deveExibirCabecalho)
    {
        if (deveExibirCabecalho)
            Notificador.ExibirCabecalho("Visualização de Revistas");

        Console.WriteLine(
            "{0, -7} | {1, -25} | {2, -6} | {3, -4} | {4, -15}",
            "Id", "Título", "Edição", "Ano", "Caixa"
        );

        Revista?[] revistas = repositorioRevista.SelecionarTodos();

        for (int i = 0; i < revistas.Length; i++)
        {
            Revista? r = revistas[i];

            if (r == null)
                continue;

            Console.Write("{0, -7} | ", r.Id);
            Console.Write("{0, -25} | ", r.Titulo);
            Console.Write("{0, -6} | ", r.NumeroEdicao);
            Console.Write("{0, -4} | ", r.Ano);

            string corSelecionada = r.Caixa.Cor;

            if (corSelecionada == "Vermelha")
                Console.ForegroundColor = ConsoleColor.Red;

            else if (corSelecionada == "Verde")
                Console.ForegroundColor = ConsoleColor.Green;

            else if (corSelecionada == "Azul")
                Console.ForegroundColor = ConsoleColor.Blue;

            Console.Write("{0, -10}", r.Caixa.Cor);

            Console.ResetColor();
            Console.WriteLine();
        }

        if (deveExibirCabecalho)
        {
            Console.WriteLine("---------------------------------");
            Console.Write("Digite ENTER para continuar...");
            Console.ReadLine();
        }
    }

    private Revista ObterDadosCadastrais()
    {
        Console.Write("Digite o título da revista: ");
        string? titulo = Console.ReadLine();

        Console.Write("Digite o número da edição: ");
        int numeroEdicao = Convert.ToInt32(Console.ReadLine());

        Console.Write("Digite o ano de lançamento: ");
        int ano = Convert.ToInt32(Console.ReadLine());

        VisualizarCaixas();

        string? idCaixa;

        do
        {
            Console.Write("Digite o id da caixa em que deseja guardar a revista: ");
            idCaixa = Console.ReadLine();

            if (!string.IsNullOrWhiteSpace(idCaixa) && idCaixa.Length == 7)
                break;
        } while (true);

        Caixa? caixaSelecionada = repositorioCaixa.SelecionarPorId(idCaixa);

        return new Revista(titulo, numeroEdicao, ano, caixaSelecionada);
    }

    private void VisualizarCaixas()
    {
        Console.WriteLine("---------------------------------");


        Console.WriteLine(
            "{0, -7} | {1, -20} | {2, -10} | {3, -20}",
            "Id", "Etiqueta", "Cor", "Tempo de Empréstimo"
        );

        Caixa?[] caixas = repositorioCaixa.SelecionarTodos();

        for (int i = 0; i < caixas.Length; i++)
        {
            Caixa? c = caixas[i];

            if (c == null)
                continue;

            string corSelecionada = c.Cor;

            if (corSelecionada == "Vermelha")
                Console.ForegroundColor = ConsoleColor.Red;

            else if (corSelecionada == "Verde")
                Console.ForegroundColor = ConsoleColor.Green;

            else if (corSelecionada == "Azul")
                Console.ForegroundColor = ConsoleColor.Blue;

            Console.WriteLine(
                "{0, -7} | {1, -20} | {2, -10} | {3, -20}",
                c.Id, c.Etiqueta, c.Cor, c.TempoEmprestimo
            );

            Console.ResetColor();
        }

        Console.WriteLine("---------------------------------");
    }
}
```

## Requisito 04: Integração na `Program`

### O que muda no sistema principal?

Agora o sistema possui dois módulos:

```
1 - Gerenciar caixas
2 - Gerenciar revistas
```

```cs
RepositorioCaixa repositorioCaixa = new RepositorioCaixa();
RepositorioRevista repositorioRevista = new RepositorioRevista();

TelaCaixa telaCaixa = new TelaCaixa(repositorioCaixa);
TelaRevista telaRevista = new TelaRevista(repositorioRevista, repositorioCaixa);

Caixa caixa = new Caixa("Relíquias", "Vermelha", 3);
repositorioCaixa.Cadastrar(caixa);

Revista revista = new Revista("Action Comics", 50, 1976, caixa);
repositorioRevista.Cadastrar(revista);

while (true)
{
    Console.Clear();
    Console.WriteLine("---------------------------------");
    Console.WriteLine("Clube da Leitura");
    Console.WriteLine("---------------------------------");
    Console.WriteLine("1 - Gerenciar caixas de livros");
    Console.WriteLine("2 - Gerenciar revistas");
    Console.WriteLine("3 - Gerenciar amigos");
    Console.WriteLine("4 - Gerenciar empréstimos");
    Console.WriteLine("S - Sair");
    Console.WriteLine("---------------------------------");
    Console.Write("> ");
    string? opcaoMenuPrincipal = Console.ReadLine()?.ToUpper();

    if (opcaoMenuPrincipal == "S")
    {
        Console.Clear();
        break;
    }

    while (true)
    {
        string? opcaoMenuInterno = string.Empty;

        if (opcaoMenuPrincipal == "1")
        {
            opcaoMenuInterno = telaCaixa.ObterEscolhaMenuPrincipal();

            if (opcaoMenuInterno == "S")
            {
                Console.Clear();
                break;
            }

            if (opcaoMenuInterno == "1")
                telaCaixa.Cadastrar();

            else if (opcaoMenuInterno == "2")
                telaCaixa.Editar();

            else if (opcaoMenuInterno == "3")
                telaCaixa.Excluir();

            else if (opcaoMenuInterno == "4")
                telaCaixa.VisualizarTodos(deveExibirCabecalho: true);
        }

        else if (opcaoMenuPrincipal == "2")
        {
            opcaoMenuInterno = telaRevista.ObterEscolhaMenuPrincipal();

            if (opcaoMenuInterno == "S")
            {
                Console.Clear();
                break;
            }

            if (opcaoMenuInterno == "1")
                telaRevista.Cadastrar();

            else if (opcaoMenuInterno == "2")

            else if (opcaoMenuInterno == "3")
                telaRevista.Excluir();

            else if (opcaoMenuInterno == "4")
                telaRevista.VisualizarTodos(deveExibirCabecalho: true);
        }

        else if (opcaoMenuPrincipal == "3")
        {

        }

        else if (opcaoMenuPrincipal == "4")
        {

        }
    }
}
```

### Organização do fluxo

O programa passa a:

1. escolher o módulo
2. delegar para a tela correspondente

```cs
TelaCaixa telaCaixa
TelaRevista telaRevista
```

### Compartilhamento de dados

```cs
TelaRevista telaRevista = new TelaRevista(repositorioRevista, repositorioCaixa);
```

👉 Ambos os módulos utilizam o mesmo repositório de caixas

### Por que isso é importante?

Isso garante que:

- revistas utilizem caixas reais
- não existam duplicações
- os dados permaneçam consistentes
