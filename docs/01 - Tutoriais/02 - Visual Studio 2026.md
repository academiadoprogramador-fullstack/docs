O Visual Studio é um ambiente de desenvolvimento integrado (IDE) criado pela Microsoft. Em termos simples, ele é um programa que reúne em um só lugar tudo o que você precisa para escrever código, compilar, depurar, testar e publicar aplicações.

Diferente do .NET SDK, que é apenas o “motor” e as ferramentas de linha de comando, o Visual Studio é a “oficina completa”. Ele fornece interface gráfica, editor de código com IntelliSense (autocompletar inteligente), depurador visual, integração com Git, designers visuais (como para interfaces web ou desktop), gerenciamento de pacotes NuGet e integração com Azure.

## SDK

Quando você instala o Visual Studio e seleciona uma carga de trabalho como “Desenvolvimento ASP.NET e Web” ou “.NET Desktop Development”, o instalador automaticamente inclui o .NET SDK correspondente.

Ou seja: você não precisa baixar o SDK separadamente se estiver usando o Visual Studio para desenvolver aplicações .NET.
